# Expense Tracker App

A React Native (Expo) mobile app to track income and expenses, with a category-based expense chart and recent transaction history.

## Features

- Add **income** and **expense** transactions
- View totals for income, expenses, and balance
- See a simple chart of expenses grouped by category
- Browse recent transactions with notes

## Run locally

```bash
npm install
npm run start
```

Then open with Expo Go on your phone, or run on an emulator.

## Tech stack

- React Native
- Expo
